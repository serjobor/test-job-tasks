   # Bitcoin Exchange Rates App
   
   ## Описание
   Приложение для отображения курсов BTC к различным валютам через CoinMarketCap API
   
   ## Технологии
   - Frontend: React + TS + Vite + Ant Design + Axios
   - Backend: Node.js + Express + Axios
   
   ## Установка
   1. Установите зависимости: `npm install` в папках client и server
   2. Создайте .env файл в папке server с API_KEY=your_key
   3. Запустите сервер: `npm run dev` в папке server
   4. Запустите клиент: `npm run dev` в папке client
